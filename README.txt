After creating a ModelSim project, 
the components unit must be added to the project. 
- Badji_ALU.vhd
- Badji_Alu_Control.vhd
- Badji_Control_Unit.vhd
- Badji_CPU.vhd
- Badji_Data_Memory.vhd
- Badji_Instruction_Memory.vhd
- Badji_Mux_2_1.vhd
- Badji_Register_File.vhd
- Badji_TB_CPU.vhd

After compiling the individual VHDL files,  Click on simulate on the menu bar. 
start simulation -> work -> Badji_TB_CPU.vhd. In the waveform window, run the simulation 
for 570 ns.
